# Recipe
레시피 제공 시스템


본 프로젝트는 1인 가구 증가에 따른 현실적인 문제를 해결하기 위해 기획되었다. 국가지표체계에 따르면 2000년 15.5%에 불과했던 1인 가구 비율이 2023년에는 35.5%로 급격히 증가했다(국가지표체계 | 지표상세정보). 특히 대학생의 경우, 한정된 예산으로 인해 배달음식보다는 직접 요리하는 경우가 많다.
하지만 자취하는 학생들을 관찰해 보면, 냉장고 안에 다양한 재료가 있음에도 불구하고 항상 동일한 요리만 반복하는 경우가 많다. 이는 새로운 요리를 시도하기 어려운 이유도 있지만, 남아 있는 재료로 만들 수 있는 요리를 찾는 과정이 번거롭기 때문이다.
이에 따라 본 프로젝트에서는 냉장고 속 남아 있는 재료를 활용하여 적절한 레시피를 추천해주는 프로그램을 개발하고자 한다. 이를 통해 자취생 및 1인 가구가 보다 효율적으로 식재료를 활용하고 다양한 요리를 시도할 수 있도록 돕는 것이 목표이다.
